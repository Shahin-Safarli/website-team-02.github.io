Website Project

Overview

In this task, we were asked to create a website about our team and our projects.

Website had to consist of at least 4 pages including this page.

We were asked to use HTML, CSS, and JS technologies to create our web pages.

Additionally, we had the option to host our website on Github Pages online platform for bonus.

Finally, we had to write a short report about our work including our Individual Contribution Table.

Structure

For the structure, we mostly used HTML (Hyper Text Markup Language).

We created a simple navigational structure to ease the access from one web page to another.

Additionally, we used a number of 'div's in our structure as different layouts such as Grid Layout and Box Layout.

Additionally, as we needed textual content on our web pages, we used various tags such as 'table', 'p', 'h1-6' etc.

Finally, we added multimedia objects to our website such as videos, pictures etc.

Design

For designing our website, we used CSS (Cascading Style Sheets).

With the help of CSS, we changed various factors which contribute to the overall looking of our website.

We, first, upgraded our navigation bar visually to present a better experience to our users, you.

Additionally, we changed the style of our 'div' elements to, at least, somehow fit the standards of nowadays' modern visuals.

Finally, we made our last touches in terms of design, such as font, color, size of elements.

Dynamism

For having some dynamism and to have some activity, we used JS (Java Script).

On our home web page, we created a greeting label depending on time.

We used conditionals to basically, condition our cases.

Additionally, for getting current time and date, we used certain built-in functions of Java Script.

Finally, we connected to our script to our label which is tagged as header in HTML.
